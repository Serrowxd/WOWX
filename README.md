# WOWX

Proof of concept for an Electron app that tracks user data - reference from the VRCX project

Planned Features:

- Character Tracker (Main / Alts)

  - Great Vault
  - IO Score
  - Keys this week / Keys this season
  - Gear
  - Raid Lockout
  - Current Key
  - Time Online

- Friend Tracking

  - Great Vault
  - IO Score
  - Keys this week / Keys this season
  - Gear
  - Raid Lockout
  - Current Key
  - Time Online
  - Instance Tracker

- Personal Trackers

  - Last seen
  - Time spent together (party / raid tracker)
  - Notes
  - Raid Team Tracker
  - Guild Tracker
